Node

node.js 是一个基于谷歌出品的js引擎（V8），并在其上定制了一套易于使用的标准类库、模块系统的Web开发平台。

node包含

  • JavaScript 语言（语法、库）
  • Node 标准库

JavaScript

和其他语言一样，它有基本的数据类型，还有函数、对象和数组这样的复杂的数据类型

同样和其他语言一样，它有一套不算丰富的类库，可能没有像Java那样丰富，但是也比c语言要丰富很多

我们可以给归类，它是弱类型动态语言

但是，它有一个致命却幸运的问题，它没有包含可以输入输出的类库

Node

正是因为js本身没有输入输出，所以node有了一个选择js作为它语言的一个理由

node本身只是一套类库，披在在了谷歌浏览器的js引擎上面
